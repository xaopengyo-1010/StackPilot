# QQ 群首次介绍

大家好，我最近在做一个小工具 StackPilot，想找几位愿意跑一下命令行的朋友帮忙测测。

它的目标是：读取本机公开硬件和环境信息，然后生成适合当前电脑的应用推荐、风险提示和可审查安装计划。当前版本不会自动安装软件，也不会修改系统设置。

现在最想验证的是 v0.4 的 GPU 检测：

- 能不能正确区分核显、独显、虚拟显卡和未知显卡；
- 会不会把共享内存误当成独立显存；
- 多显卡机器上 primary GPU 是否选得合理；
- 没有 NVIDIA 显卡时，CUDA / NVIDIA 相关提示是否足够清楚。

可以运行：

```bash
python -m stackpilot scan
python -m stackpilot recommend --goal comfyui_starter
```

如果愿意反馈，直接贴关键输出就行。发之前可以打码用户名、路径、序列号和任何私人信息。

我更需要真实机器上的“不准”和“不好理解”，不是刷数据。谢谢。
