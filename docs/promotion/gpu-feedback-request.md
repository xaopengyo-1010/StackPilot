# GPU 检测反馈请求

我想找几台不同配置的电脑测试 StackPilot v0.4 的 GPU 检测。

当前最需要的机器类型：

- 只有 Intel 核显；
- 只有 AMD 核显，比如 Radeon 780M；
- NVIDIA 独显笔记本；
- AMD 核显 + NVIDIA 独显；
- 台式机独显；
- 远程桌面、虚拟机、云电脑或带虚拟显示驱动的环境；
- 显卡名称比较新的设备。

运行命令：

```bash
python -m stackpilot scan
```

重点看这几项：

- `检测到的 GPU`
- `主要性能判断 GPU`
- `GPU 选择原因`
- `显存置信度`

希望反馈：

- GPU 列表是否完整；
- 核显 / 独显 / 虚拟显卡是否分类正确；
- primary GPU 是否选对；
- 共享内存是否被误写成独立显存；
- 输出文案是否能看懂。

请不要贴隐私信息。可以打码用户名、路径、序列号、账号和私人文件名。

StackPilot 当前不会自动安装软件，也不会修改系统设置。
