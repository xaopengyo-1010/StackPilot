# QQ Group Feedback

This file records early feedback from QQ groups, open-source chat groups, private messages, and Issues.

本文件用于记录来自 QQ 群、开源交流群、私聊和 Issue 的早期反馈。它只记录可用于修正 StackPilot 的技术信息，不记录个人隐私。

## Feedback Log

| Date       | Source   | Machine / Environment | Feedback                   | Priority | Status | Notes                              |
| ---------- | -------- | --------------------- | -------------------------- | -------- | ------ | ---------------------------------- |
| YYYY-MM-DD | QQ group | Unknown               | Placeholder feedback entry | P2       | open   | Replace with real feedback later. |

## Priority

- P0: blocks basic usage or causes wrong safety assumptions.
- P1: affects detection, recommendation, or install plan quality.
- P2: documentation, wording, or usability improvements.

## Privacy Rule

Do not record personal names, QQ numbers, private paths, serial numbers, exact addresses, or sensitive device identifiers.

不要记录个人姓名、QQ 号、私人路径、设备序列号、精确地址或敏感设备标识。提交反馈前请先打码用户名、个人目录、账号、密钥和私人文件名。

## Status Values

- `open`: 等待确认或修复。
- `triaged`: 已确认优先级和处理方向。
- `fixed`: 已修复并有测试或文档证据。
- `wontfix`: 暂不处理，需要写明原因。
